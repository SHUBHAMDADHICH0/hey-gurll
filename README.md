# welcome to my project name - Hey-Gurll

Hey Gurll is a small web project made for sending cute and special wishes.

It can be used when:
- you want to wish someone on their birthday
- you want to say Happy Women's Day
- someone is angry and you want to make them smile again
- or you just want to send a sweet message

The idea of this project is to make wishing someone more fun and interactive.
